但是，修改成功后的提交，只对未来的提交有效；这里必须得吐槽一下gitee和github统计提交的不同之处

  • gitee统计提交依据的是用户名+邮箱，只要有一个不对就无法统计
  • github统计提交只依据邮箱，只要你邮箱对了，用户名是不管你的

所以要想让gitee的统计能生效，那就必须要修改之前所有提交信息中的用户名和邮箱
