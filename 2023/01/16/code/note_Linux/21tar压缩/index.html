1.起因

今天下午写阿狸bot的代码的时候，写错了aiofiles的保存操作

注意这两个函数，一个是json.dump；一个是json.dumps

就是因为这里写错了，导致bot执行全局变量保存的时候，所有文件都没有成功保存——还直接把本地文件清空了！！！

当时的我是真的欲哭无泪啊，都不知道如何是好
